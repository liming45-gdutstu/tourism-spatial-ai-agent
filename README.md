# tourism-spatial-ai-agent

## 项目简介

面向国土空间规划文旅垂直场景的AI智能体产品，主打游客行程智能规划、景区空间推荐、多模态文旅内容生成。

## 项目职责（AI产品实习简历复用）

1. 需求设计：梳理文旅游客核心痛点，输出完整PRD、用户画像、竞品分析
2. Agent搭建：基于Coze搭建空间智能体工作流，拆解意图识别、工具调用、多轮对话链路
3. 提示词工程：搭建三级分层提示词模板，完成文生图/行程规划多模型调优
4. 数据处理：高德POI空间数据批量采集、清洗、空间可视化分析
5. 原型落地：输出产品交互原型、功能迭代台账、线上项目复盘文档

## 工具栈

Coze智能体、高德地图API、Git版本管理、Figma原型、多LLM大模型、ArcGIS空间数据

## 项目产出

产品需求文档、Agent工作流截图、提示词资产库、空间数据脚本、交互原型图
