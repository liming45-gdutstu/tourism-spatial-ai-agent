# tourism-spatial-ai-agent
多代家庭文旅GIS AI行程规划Agent · PC端MVP产品作品集

## 一、项目简介
面向珠三角三代同游家庭决策者打造的轻量化AI行程规划产品，解决当前OTA文旅AI普遍存在的三大痛点：老人步行负荷超标、缺少儿童适配景点、行程跨片区远距离折返；
创新引入**GIS Kmeans片区聚类算法+多代人群体力加权约束体系**，量化步行上限、季节人流、预算、配套设施四大硬性条件，输出贴合老中青全年龄段家庭的可落地短途文旅行程。

## 三、核心产品差异化亮点
1. 双层用户约束体系：UserProfile出行画像录入 + PreferenceConfig人群偏好配置，细分老人/儿童年龄段、步行耐受、兴趣标签、预算档位，前端实时拦截空值、负数非法输入；
2. GIS空间量化核心能力：自动聚类游玩片区、计算日均步行距离、区分旺季人流热度，色块地图可视化展示片区边界；
3. 全闭环用户操作流程：采集出行参数 → GIS生成片区行程草案 → 修改重生成 → 确认导出门户文字攻略；
4. 家庭友好配套模块：景点详情独立弹窗，内置5分钟步行可达本地美食、景区适老化/儿童配套、旺季避坑提示。

## 四、完整业务链路
### 前端用户操作流程
首页参数录入页 → 多代偏好配置页 → 片区地图行程总览页 → 点击景点弹出详情弹窗 → 返回修改参数/导出完整攻略
### 后端算法计算流程
高德POI数据采集清洗 → Kmeans片区聚类分区 → 步行负荷/预算/营业时间三重校验 → 分层Prompt生成结构化家庭行程

## 五、四大原型页面功能说明
1. 参数录入页：填写目的地、出行天数、交通方式、成人/老人/儿童人数及年龄分层，输入异常实时红色提示；
2. 偏好配置页：设置步行体力上限、景点兴趣、预算档位，支持返回修改基础出行信息；
3. GIS行程总览页：聚类片区可视化地图、横向景点实拍卡片、四大GIS量化指标、配套本地美食推荐；底部支持重生成、确认行程、导出攻略；
4. 景点详情弹窗：全景景点图、人群适配评分、周边美食、出行避坑提示，右上角关闭按钮。
